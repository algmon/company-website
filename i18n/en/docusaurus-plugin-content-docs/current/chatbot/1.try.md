# 马上体验
## 第一步 扫码
扫描下方二维码即体验我们的智能聊天机器人AI熊猫Rita，Rita是一个教培助手，她能友好地和同学聊天，明晰其知识点疏漏并为之提供相应教育建议和答疑。

![](./img/suanfamama.core.product.02.png)

## 第二步 畅聊
请使用以下格式和聊天机器人开始畅聊，如：
1. "bot 搜索 双减教育政策"
2. "bot 画 一对猫 仰望星空"

![](./img/chat.interface.1.png)
